  • General purpose of the research:

The project's general objective is to develop a visual odometry algorithm using Voxel mapping, which can identify the location of a mobile robot in external environments. The development of the project will be carried out using an autonomous mobile platform called Warthog, and using equipment and materials available at the Robotics and Autonomous Systems Laboratory. The environment used for tests is the patio of Cimatec 4, in which the mobile robot must perform the task of locating itself in the environment through information obtained by the camera.
